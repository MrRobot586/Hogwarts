How I built it

The application is divided into 3 main sections in which a list of the characters belonging to the saga of Harry Potter books is shown, as appropriate. There is a section for students, one for teachers and one where all the characters are shown where we can filter the list according to the house of wizardry we select in a navigation menu.
Internally, the application consists of 7 key components, which are displayed depending on the page or section in which the user is, using a service to obtain the data of the characters of a public api, which are displayed on the screen in individual cards, where certain characteristics of each character and their photograph are shown.
In the students section there is a button through which we can navigate to a new section in which we can apply to join the student body of Hogwarts school, in this section we can fill out a form and see the result in real time, once the form is sent then we can see our application and the status in which it is (this data is stored locally).
Each component is built with bootstrap for the CSS styles, some of them have icons (using the font-awesome package) and some of them are static like the navigation bar and the footer. I also added some effects and transitions to make the application dynamic and pleasing to the eye, almost all the elements are responsive (i.e. they change according to some data). Note, the application is in English to demonstrate my level in the language, but I also used the translator DeepL to make some translations, if not most of them.
